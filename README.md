# BadUSB
A diy usb pin cracker

Target: Any MCU that has a naitive USB (Make sure to download the board package, dont change the code, complier handles everything)

total combinations: 10,000 (0000 - 9999)

NOTICE: this wont work on new devices because they limit the amount of “tries” you can input.

To increase or decrease the speed edit the delay on line 57 of the code.
